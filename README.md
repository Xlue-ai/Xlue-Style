# Xlue-Style

> **Xlue Design System** — Official brand colors, typography, components, and layout specifications for all Xlue products.

## Contents

| File | Description |
|------|-------------|
| [`index.html`](./index.html) | Interactive static style guide webpage |
| [`style.md`](./style.md) | Comprehensive design system documentation (Markdown) |

## Quick Start

Open `index.html` directly in any browser — no build step required.

```bash
open index.html
# or
python3 -m http.server 8080
```

## Brand At a Glance

| Token | Value | Usage |
|-------|-------|-------|
| `--color-primary` | `#007D7D` | Buttons, links, accents |
| `--bg-l0` | `#FAFAF8` | Body background |
| `--bg-l1` | `#EFF4F4` | Cards, sections |
| `--text-primary` | `#0D2B2B` | Headings, body copy |
| `--border-default` | `#D8E2E0` | Dividers, input borders |
